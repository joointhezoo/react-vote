## React-vote
> This project was bootstrapped with Create React App.

## setup
### prerequisites
- node 설치
- npm 또는 yarn 설치 

### development
```markdown
$ git clone https://github.com/joointhezoo/react-vote.git
$ cd react-vote

# with yarn

$ yarn install
$ yarn start


# with npm

$ npm install
$ npm start
```
